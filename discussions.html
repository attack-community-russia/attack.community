ДИСКУССИИ


Важные для сообщества вопросы обсуждаются публично в формате научных дискуссий. Для таких обсуждений создаются отдельные группы в Telegram — Дискуссионные Комнаты (ДК).

Результат

Итог каждой дискуссии публикуется в канале сообщества в формате короткой сводки.
Обсуждение остается в ДК для возможности ссылаться на отдельные сообщения внутри Telegram.
Вместе с тем, обсуждение целиком экспортируется и сохраняется на сайте сообщества.

Принять участие

Для получения возможности отправлять сообщения в ДК необходимо пройти верификацию.
Для читателей вход свободный.
